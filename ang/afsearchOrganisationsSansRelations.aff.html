Contexte

  • Certaines fiches Organisations n'ont pas de Relations.

Risques

  • Pas de lien clair entre les Individus et les Organisations ; Pas d'identification claire sur les Interlocuteurs appartenant à l'Organisation ; Erreurs de ciblage.

Actions à mener

  1. Cliquer sur le lien "Ajouter une relation".
  2. Dans la fenêtre qui s'ouvre, choisir le Type de Relation souhaité.
  3. Dans l'entrée Contacts, choisir les Individus concernés, et cliquer sur Enregistrer la Relation.
