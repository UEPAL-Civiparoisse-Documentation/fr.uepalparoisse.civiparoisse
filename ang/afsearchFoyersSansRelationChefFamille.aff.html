Contexte

  • Certaines fiches Foyer n'ont pas de Relation Chef de Famille.

Risques

  • Pas de lien clair entre les Individus et leur Foyer ; Pas d'identification claire sur la composition du Foyer ; Erreurs de ciblage.

Actions à mener

  1. Cliquer sur le lien "Ajouter une relation".
  2. Dans la fenêtre qui s'ouvre, choisir le Type de Relation "a pour Chef de Famille".
  3. Dans l'entrée Contacts, choisir le ou les Individus concernés.
  4. Cliquer sur "Enregistrer la Relation".
