Contexte

  • Certaines fiches Foyer n'ont pas de Relations Membre du Foyer.

Risques

  • Pas de lien clair entre les Individus et leur Foyer ; Pas d'identification claire sur la composition du Foyer ; Erreurs de ciblage.

Actions à mener

  1. Cliquer sur le lien "Ajouter une relation".
  2. Dans la fenêtre qui s'ouvre, choisir le Type de Relation "a pour Membre du Foyer".
  3. Dans l'entrée Contacts, choisir les Individus concernées (ajouter autant de relations qu'il y a de personnes dans le Foyer).
  4. Cliquer sur "Enregistrer la Relation".
